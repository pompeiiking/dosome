忽略文件
在主目录下建立”.gitignore“文件，文件有以下规则：
1.忽略文件中的空行或以井号#开始的行会被忽略
2.如果名称的最前面有一个感叹号（!)，表示例外规则，不被忽略
3.如果名称的最前面是一个路径分隔符（/），表示要忽略的文件在此目录下，而子目录中的文件不被忽略
4.如果名称的最后面是一个路径分隔符（/），表示要忽略的文件在此目录下该名称的子目录，而非文件

忽略所有.txt结尾的文件，”*.text“这样上传不会被选择
